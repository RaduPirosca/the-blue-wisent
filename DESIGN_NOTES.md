# The Blue Wisent: Visual Direction

## Direction

Forest Archive: a quiet personal library at the edge of a blue forest. The site should feel literary, grounded, and curious, with enough modern structure to support programming experiments.

## Palette

- Paper: `#f7f4ec`
- Deep paper: `#ebe4d4`
- Ink: `#1f2526`
- Blue: `#244f8f`
- Deep blue: `#17345f`
- Spruce: `#245049`
- Moss: `#7c8d5a`
- Rust accent: `#9f5135`

## Type

- Serif headings for an editorial, field-guide feeling.
- System sans-serif body for clean reading and easy maintenance.

## Site Shape

- `Writing`: essays, notes, reflections, technical posts.
- `Experiments`: LLM tools, interface sketches, simulations, future Three.js scenes.

## Future Ideas

- Replace `WisentMark.astro` with a refined logo or generated mark.
- Add an RSS feed.
- Add MDX for posts that embed interactive components.
- Add a reusable `ExperimentFrame` component for canvas and Three.js pages.
